GNOME Gmail Privacy Policy.

GNOME Gmail requires permission to create draft messages on Gmail, and, if possible, determine a valid URL for user interaction with that draft.

No personal information is shared outside of this process. This includes user information, including email address, and all authentication tokens.

GNOME Gmail does not do ads.
